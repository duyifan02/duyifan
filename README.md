# Personal Homepage

A modern, Google-inspired personal homepage template with a bilingual (中文 / English) toggle.

## Features

- Clean, modern UI style with card-based layout
- Language switch button (`中文 / EN`) for instant bilingual content toggle
- Responsive design for desktop and mobile
- Sections ready for research-oriented profile content

## Files

- `index.html` — page structure
- `styles.css` — visual styling
- `script.js` — bilingual toggle and text rendering

## Quick Start

Open `index.html` directly in browser, or run a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
